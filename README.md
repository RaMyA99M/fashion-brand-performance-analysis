# Fashion Retail Brand Performance Analysis

## Overview

This project analyzes fashion retail brands to identify inventory inefficiencies and revenue loss opportunities.

Using stockout rates, average selling prices, and estimated lost revenue, brands are segmented into strategic categories to support pricing and inventory decisions.

## Business Problem

Retailers often struggle to balance inventory availability with demand.

Frequent stockouts can result in:
- Lost sales
- Reduced customer satisfaction
- Revenue leakage

This project investigates how pricing and stockout performance vary across brands and identifies areas for operational improvement.

## Objectives
- Measure stockout performance by brand
- Estimate lost revenue from inventory shortages
- Compare pricing strategies across brands
- Segment brands into actionable business categories
- Generate data-driven recommendations
